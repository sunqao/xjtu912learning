# 多元函数微分学

## 多元函数的极值点，驻点问题

能够使得$f'_x(x, y) = 0, f'_y(x, y) = 0$的点就是驻点，极值点一定是驻点，但是驻点不一定是极值点

所以极值点的求法是根据$f'_x(x, y) = 0, f'_y(x, y) = 0$求出所有的驻点，然后根据极值点的充分条件判别法则
