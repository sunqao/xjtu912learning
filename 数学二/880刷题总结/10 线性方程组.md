# 基础题

## 一

### n - r(A)的理解

对于AX = 0这样一个齐次方程组，r(A)是化作阶梯型之后的非零行的个数，也就是AX = 0的有效方程的个数，同时也是待定未知数的个数

> 待定未知数就是用其他的未知数表示的未知数

n - r(A)中n是未知数的个数，J(A)  = n - r(A)就是基础解系的极大无关组中向量个数，也就是基础解系中解向量的个数，也是自由未知数的个数

> 自由未知数就是用来表示待定未知数的个数，自由未知数可以随便赋值来求待定未知数，可以让自由未知数轮流取1，其他取0来强制得到方程的基础解系

### （1）非齐次方程组的解的结构

AX = b这样的非齐次方程的解就是齐次方程AX = 0的通解加上一个AX = b的特解

### （2）矩阵转置的解的情况$A^TAx = 0 \to Ax = 0$

这里主要来证明$A^TAx = 0 \to Ax = 0$

$A^TAx = 0$两边同时乘x^T，有：$(Ax)^TAx = 0$，而Ax是一个列向量，所以这个式子其实是列向量Ax = 0的内积，因此Ax = 0

### （3）$A^*$的处理 + $A^*$与A的秩的关系

> 关于$A^*$其实有如下几种常见的处理方式：
>
> **（1）行列式的值**
>
> $|A^*|=|A|^{n - 1}$
>
> **（2）一个常见的等式：**
>
> $AA^* = A^* A = |A|E$，不过注意伴随矩阵再取一次伴随不是A本身
>
> **（3）秩相关：**
>
> 如果$r(A) = n$，则$r(A^*) = n$
>
> 如果$r(A) = n - 1$，则$r(A^*) = 1$
>
> 如果$r(A) <= n - 2$，则$r(A^*) = 0$
>
> **（4）特征值相关**
>
> 一般而言题目中给出了$A^*$的相关信息，我们就得转化为A的信息，如果一个特征向量是A的特征向量有$A\alpha = \lambda \alpha$，则这个特征向量也是$A^*$的特征向量，并且有：$A^* \alpha = \frac{|A|}{\lambda} \alpha$，$A^*$的特征值会发生变化

这道题目，因为任意的n维向量都是$A^*$的解，所以我们一定可以找到n个无关的向量是$A^*$的解，那么其解向量就有n个，因此$n - r(A^*) = n$，所以$r(A^*) = 0$，其实A的伴随就是一个零矩阵

我们根据这个再分析A的情况就得到结果了

### （4）$AB = 0$条件的使用

对于AB = 0有两个思考方向

**（1）秩相关**

有$r(A) + r(B) \le n$，这里的n是A，B两个矩阵共有的部分，即A的列数和B的行数

**（2）转化成方程形式**

$AB = [A\beta_1, A\beta_2, ...A\beta_n] = 0$，这里用矩阵乘法的本质来理解，AB乘积的矩阵的列向量是A的列向量的线性组合，组合系数就是B的每个列向量，所以这里B的每个列向量就是AX = 0的解

这道题目用第一个角度比较容易做，这里不再赘述

用第二个角度，如果$\lambda = -2$，那么r(A) = 3，对于AX = 0未知数的个数n = 3 = r(A)，因此AX = 0只有零解，所以B为零矩阵，显然与题目矛盾，因此$\lambda = 1$，此时r(A) = 1，解向量的个数为J(A) = n - r(A) = 2，也就是说组成B的列向量的向量最多有两个最大无关组组合在一起，r(B) <= 2，因此B的行列式就是等于0

### （5）方程组解的问题的总结

**（1）如果是齐次方程组AX = 0**

这种方程组是始终有解的，即零解，当r(A) = n的时候有唯一解，当r(A) < n的时候无穷解

**（2）如果是非齐次方程组AX = b**

如果增广矩阵r(A|b) > r(A) 则无解，出现0 = b1(b1 != 0)这样的方程

其他情况一定有解

如果r(A|b) = r(A) = n，则只有唯一解，系数矩阵如果是方阵，其行列式一定不等于0

如果r(A|b) = r(A) < n，则有无穷解，系数矩阵如果是方阵，其行列式一定等于0

正常情况下对这种题目，最好不要直接对增广矩阵直接进行行变换化简，而是直接讨论方阵A的行列式的情况，因为直接化简会有很多的参数，但是通过直接对行列式讨论可以直接得到A中参数的分界点

### （6）AB的矩阵的秩的性质

$r(AB) \le \min[{r(A), r(B)}]$

## 三

### （3）基础解系和通解的规范 + 求两个方程的公共解的方法 + 公共解与同解

基础解析其实就是组成通解的解向量

通解需要在基础解析前面加上组合系数k1, k2之类的

**求公共解的方法：**

**（1）直接联立两个方程组**

这种方式直接将两个方程组变成一个方程组直接解出通解

**（2）求出某一个方程的通解，然后代入另一个方程得到解向量的系数关系**

公共解包含的解一定小于某个方程的通解，即比通解多了一些系数限制，这个限制就是第二个方程的限制，所以我们直接代入第二个方程组得到解向量系数之间的关系，就得到限制了

**（3）求出两个方程组的通解，然后令两边的通解相等，得到系数之间的关系**

跟第二点差不多，还是得得到某个方程组通解的系数关系，也就是加一个约束

**AX = b与BX = c的公共解和同解：**

公共解就是即满足第一个方程组的解又满足第二个方程组的解，两个方程有公共解不意味着两个方程的解完全一样，求公共解的方法就是如上三种

同解是两个方程组有完全一样的解，这个条件比公共解严格，初等行变换就是同解变换，两个方程组同解，那么其行向量组一定等价，并且系数矩阵也互相等价

### （5）同解的处理，一个易错点

这道题的第二问直接将第一问的特解代入就可以得到a, b, c的值

不过这里的a, b, c只是建立在两个方程有公共解的条件下求出来的（我们用了一次公共解，然后得到了这个结果）；所以最后需要验证一下，因此还要将a, b, c代入第二个方程后求出第二个方程的通解，发现与第一个方程的通解一样，这样才能说明a, b, c取这些值的时候同解

### （6）$A^*x = 0$的处理 + r(A)与$r(A^*)$的关系

**$A^*x = 0$的处理：**

关于这个式子，我们可以用一个三阶的来帮助理解

对于其第一个方程，$A_{11}x_1 + A_{21}x_2 + A_{31}x_3 = 0$，可以发现A的第一列向量就是$A^*x = 0$的第一行方程的解

同理A的第二列向量就是$A^*x = 0$的第二行方程的解，A的n个向量分别是$A^* x = 0$的n行方程的解

所以下一步猜想A的一个极大无关组是不是就是$A^* x = 0$的通解

但是注意每个A的列向量只是对应$A^* x = 0$每行方程的解

**而$AX = 0$通解（基础解系）需要满足下面三个条件：**

（1）还要求每个向量之间极大无关

（2）这个极大无关组的向量个数等于n - r($A$)

（3）这个向量组的每个向量都是$A x = 0$的解

所以下一步就需要考察r($A^*$)

 **r(A)与$r(A^*)$的关系：**

如果$r(A) = n$，则$r(A^*) = n$

如果$r(A) = n - 1$，则$r(A^*) = 1$

如果$r(A) <= n - 2$，则$r(A^*) = 0$

通过题意我们可以得到$r(A^*) = 1$且$r(A) = n - 1$，所以$A^* x = 0$的每行方程都是等价的，因此A的每个列向量都是$A^* x = 0$的解，且$A^* x = 0$通解的个数是n - 1

所以A的列向量的一个极大无关组，就是$A^* x = 0$的基础解系了；因为A11不为0，那不就是除了第一列之外的列向量线性无关答案直接出来了（A11!=0说明去除了第一行第一列剩下的矩阵的行列式不为零，那么这些列向量就是线性无关的）

当然也可以直接根据$A^*x = 0$的第一行方程直接进行构造

### （5）Ax = 0和$A^TAx = 0$的关系

首先，这两个方程组一定是同解的，可以通过第二个方程左乘一个x转置得到

**所以我们可以得到一个很重要的结论：**
$$
r(A)=r(A^T) = r(A^TA)
$$
那这道题目，显然我们需要证明：
$$
r(A^TA) = r(A^TA|A^Tb)
$$
证明这个东西，一般就是根据不等式大于等于，并且小于等于某个数得到

首先右边相等于左边加了一列，那么显然有$r(A^TA) \le r(A^TA|A^Tb)$

而右边可以写成$A^T (A|b) = A^TA|A^Tb$根据矩阵乘法$r(A^TA|A^Tb) \le r(A^TA)  $

就得到证明了

### （8）满秩矩阵乘法的性质

**满秩矩阵有下面两个性质：**

（1）如果A列满秩，则r(AB) = r(B)；如果B行满秩则r(AB) = r(A)，即一个矩阵乘了一个满秩矩阵其秩不变

（2）如果A是列满秩的，ABX = Ab与BX = b同解

> 这是因为A列满秩，因此其行秩（有效方程个数）等于列秩等于未知数的个数，所以AY = Ab只有唯一解Y = b，因此BX = b，也就满足BX = b；同理BX = b的解当然满足ABX = Ab，因此两个方程组同解

这道题目就是根据满秩矩阵乘法的第二个性质得到结果

### （2）正交矩阵的性质

**正交矩阵的两点性质：**

（1）每一行（列）的模值为1

（2）不同行（列）之间互相正交

（3）正交矩阵一定是可逆矩阵，并且$A^TA = AA^T = E$
