### （1）矩阵的秩的性质 + AB = O的矩阵的秩的性质 + r(A) = 1的矩阵的性质

如果AB = O，则有r(A) + r(B) <= n，这里的n是A的列数，B的行数

在这道题目中，那就是r(A) + r(B) <= 3，由于r(B) >= 2，所以r(A) <= 1<1，但是A有一阶非零子式，所以r(A) = 1

> 秩为1的矩阵的性质
>
> （1）**任何一个列向量乘行向量得到的矩阵都是一个r = 1的矩阵**，同理任何一个r = 1的矩阵都可以分解为一个列向量乘行向量，任何一个r = 1的矩阵的每个列向量都是成比例的（这个性质用矩阵乘法的本质去理解会非常简单）
>
> （2）**关于特征值**，r = 1的矩阵A的特征值就是tr(A), 0, 0, 0.....，并且A可以写作$A = \alpha \beta^T$，其中$\alpha$是一个列向量，$\beta^T$是一个行向量，且$tr(A) = \alpha^T \beta$矩阵A的迹就是两个向量的点积
>
> （3）**关于特征向量**，r = 1的矩阵A可以写成：$\alpha\beta^T = A$，对于$\lambda = tr(A)$的特征向量，就是$\alpha$，这是因为：$A\alpha= \alpha\beta^T \alpha= tr(A) \alpha = \lambda \alpha$；对于$\lambda = 0$的特征向量，其就是AX = 0的解，这是因为特征向量$\eta$满足：$(A - \lambda E)\eta = 0$，此时$\lambda = 0$，那显然$\eta$就得满足$A\eta = 0$
>
> （4）**相似对角化问题**
>
> i 如果tr(A) != 0，那么主要就是看特征值0是否有n - 1个解向量，对于$A\eta = 0$，解向量个数就是$n - r(A) = n - 1$，刚好n - 1个，所以如果迹不为零，那么这个秩为1的矩阵是一定可以相似对角化的
>
> ii 如果tr(A) = 0，对于$A\eta = 0$，解向量个数就是$n - r(A) = n - 1$不等于0的重数，显然不能进行相似对角化
>
> > n阶矩阵A可以相似对角化，等价于A有n个线性无关的特征向量，等价于A的任意k重特征值有k个线性无关的特征向量，1重特征根一定有一个非零特征向量
> >
> > 要算一个特征值$\lambda$有多少个线性无关的特征向量的个数，由于特征向量满足方程：$(A - \lambda E)\eta = 0$，所以其实就是方程$(A - \lambda E)\eta = 0$的解向量个数

### （5）E的使用 + 隐藏条件的处理

这道题目关键是获得题目中的隐藏条件，上面有类似的题目，如果出现两个向量相乘的r = 1的矩阵的话，可以通过两边同时再乘这两个向量的相乘，然后提出一个常数构造一个新的条件

而这里题目中直接的隐藏条件有

|B| != 0，说明B可逆，并且$B^T$可逆

$(A - E)^{-1} = (B - E)^T$，说明$(B - E)^T$可逆

这道题目先用转置的运算处理一下得到$(A - E)^{-1} = (B^T - E)$，其实可以直接左右乘$(A - E)$得到一个A的式子：
$$
A = (B^T - E)^{-1} + E
$$
然后这里用隐藏条件$(B^T - E)(B^T - E)^{-1} = E$来代换掉E，接着继续化简，取逆之后就得到结果了

### （16）对方程组行列变换的理解

### （2）r = 1的矩阵的应用 + r(A)和$r(A^*)$的关系

这里的看见对角线都是一样元素的矩阵可以拆为一个B + (a - b)E，这里的B是元素全为b的矩阵，就是一个秩等于1的矩阵

**$r(A)$与$r(A^*)$的关系**

（1）$r(A) = n$的时候$r(A^*) = n$

> 这个性质根据$AA^* = |A|E$得到，两边同取行列式，得到A的伴随行列式的矩阵不为零

（2）$r(A) = n - 1$的时候$r(A^*) = 1$

> 还是根据公式$AA^*  = |A|E$得到$AA^* = 0$，此时有$r(A) + r(A^*) \le n$，得到$r(A^*) \le 1$，由于$A^*$的元素都是A的n - 1阶的代数余子式构成，而$r(A) = n -1$就说明A必存在一个代数余子式不为零，所以$r(A^*) = 1$

（3）$r(A) < n - 1$，此时$r(A^*) = 0$

> 因为此时A中不存在n - 1阶的代数余子式不为零，所以此时的伴随矩阵就是一个零矩阵，秩当然就是零了

**广义初等变换：**

所谓广义初等变换，就是在分块矩阵中对矩阵使用初等变换，并且广义初等变换也保证整个矩阵的秩不变

行变换，一个矩阵行左乘一个东西加到另一行

列变换，一个矩阵列右乘一个东西加到另一列

这种初等变换一般就只进行第三类初等变换，其实行变换的本质是左乘一个**可逆分块矩阵**，列变换的本质是右乘一个可逆分块矩阵（做题的时候稍微想想即可），比如第一个分块矩阵的行变换和列变换分别是如下乘法操作：

**一些结论：**

A和B的列向量组等价，则有三秩相等：$r(A) = r(B) = r(A | B)$

A和B的行向量组等价，则有三秩相等：$r(A) = r(B) = r(\frac{A} {B} )$

关于这个三秩相等挺好理解的，因为B的行或者列都可以被A表示，因此也可以被A消掉，并且对于AX = 0和BX = 0来说其有效方程是一样的，因此这两个方程组同解

**AB乘积的秩的性质：**

$r(AB) \le \min [r(A), r(B)]$
